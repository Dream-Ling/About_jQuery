jQuery 杂项方法

jQuery 杂项方法

方法
描述
data()
向被选元素附加数据，或者从被选元素获取数据
each()
为每个匹配元素执行函数
get()
获取由选择器指定的 DOM 元素
index()
从匹配元素中搜索给定元素
$.noConflict()
释放变量 $ 的 jQuery 控制权
$.param()
创建数组或对象的序列化表示形式（可在生成 AJAX 请求时用于 URL 查询字符串中）
removeData()
移除之前存放的数据
size()
在版本 1.8 中被废弃。返回被 jQuery 选择器匹配的 DOM 元素的数量
toArray()
以数组的形式检索所有包含在 jQuery 集合中的所有 DOM 元素
